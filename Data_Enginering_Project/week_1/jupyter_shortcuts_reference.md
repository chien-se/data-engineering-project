# Jupyter in Cursor / VS Code — shortcut reference (Windows)

Verify or remap: **Ctrl+K Ctrl+S** → search `notebook` or `jupyter`.

## Run cells

| Shortcut        | Action |
|----------------|--------|
| **Ctrl+Enter** | Run cell, stay on it |
| **Shift+Enter** | Run cell, insert new cell below, focus new cell |
| **Alt+Enter** | Run cell, insert new cell below, focus stays on current |

## Inside a cell (edit mode)

| Shortcut | Action |
|----------|--------|
| **Enter** | New line |
| **Esc** | Command mode (cell selected, no text cursor) |
| **Enter** (from command mode) | Edit cell again |

## Command mode (press **Esc** first)

| Key | Action |
|-----|--------|
| **A** | Insert cell above |
| **B** | Insert cell below |
| **dd** | Delete cell |
| **J** / **K** | Next / previous cell |
| **M** | Markdown cell |
| **Y** | Code cell |
| **Alt+↑ / ↓** | Move cell up / down |
| **L** | Line numbers (this cell) |
| **Shift+L** | Line numbers (whole notebook) |
| **z** | Undo last notebook action |

## Other

| Shortcut | Action |
|----------|--------|
| **Ctrl+S** | Save |
| **Ctrl+F** | Find (use filter for cell output vs source) |

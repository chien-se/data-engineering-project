## 🎯 Mock Interview Template

### Before Coding (3 min)
```
1. Clarify problem:
   - Ask about edge cases
   - Confirm constraints
   - Understand input format

2. Think out loud:
   - "Let me start with brute force..."
   - "Then optimize to..."
   - State time/space complexity

3. Walk through example:
   - Use simple example
   - Trace through logic
   - Find patterns
```

### During Coding (main time)
```
1. Use problem-specific template
2. Start with base cases
3. Build incrementally
4. Add comments
5. Don't test as you go - finish first
```

### After Coding (2 min)
```
1. Test with given example
2. Test edge cases:
   - Empty input
   - Single element
   - All same values
   - Very large input
3. Discuss optimization opportunities
4. Ask follow-up questions
```
